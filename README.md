# Aeriadne — Somnus Intelligence Marketplace

> **Aeriadne** is the first-of-its-kind marketplace for agentic operability. It delivers high-fidelity skills, plugins, subagent prompts, and server-plane activations to any coding agent (Codex, Claude Code, OpenCode, Grok Build).

---

## Quick Launch

Launch the marketplace index on your machine with one command:

```bash
curl -sSL https://raw.githubusercontent.com/calisweetleaf/Aeriadne/main/install.sh | bash
```

*Prerequisites: git, python3.*

---

## The Marketplace Planes

Aeriadne organizes the intelligence stack into seven navigable planes:

### 1. Plugins (`Plugins/`)
Installable package shells that provide client-specific affordances.
- **Aeriadne**: The operator and registry shell.
- **CTMv3**: Workspace activation and codebase topology.
- **Mentat**: Live session instrumentation substrate.

### 2. Skills (`Skills/`)
Reusable cognitive workflows.
- **Constitutional Prompt Framework (CPF)**: The architecture of agent identity and doctrine.
- **Grok Build Configurator**: Specialized one-skill runtime setup.

### 3. Agents (`Agents/`)
Specialized subagent prompt packs for parallel workstreams.
- `prompt-architect`, `package-cartographer`, `release-sentinel`, etc.

### 4. Servers (`Servers/`)
Catalog and reference cards for canonical tool planes like **Sovereign BB7**.

### 5. Registry (`Registry/`)
The machine-readable inventory (`plugins.yaml`, `skills.yaml`) that powers discovery and installation.

### 6. Marketplace (`Marketplace/`)
Human-facing cards and rendered indexes for browsing available assets.

### 7. Core (`Core/`)
The underlying kernels and runtimes that power the marketplace activations.
